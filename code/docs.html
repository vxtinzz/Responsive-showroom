DOCUMENTAÇÃO DO PROJETO

Análise técnica e objetivos do desenvolvimento web

Objetivos

Desenvolver uma plataforma de catálogo digital para a marca autoral Outz.Lab, integrando a vitrine de vestuário e acessórios a um showroom de serviços e drinks.

Público Alvo

Jovens e entusiastas da cultura urbana, moda streetwear e design independente, que buscam produtos exclusivos e experiências gastronômicas artísticas.

Tecnologias e Aplicação

O serviço foi desenvolvido utilizando HTML5 e CSS3 manual para a estrutura principal.

Foram utilizados recursos modernos de CSS, como Flexbox para organização dos elementos e CSS Grid para auxiliar na disposição dos conteúdos e adaptação responsiva, principalmente na estruturação vertical das páginas.

HTML5 CSS3 DESIGN RESPONSIVO

Requisitos Mínimos e Restrições

O projeto inclui uma página inicial de apresentação, catálogo de produtos, seção de notícias, sistema de cadastro, página de contato, documentação do projeto, estilização personalizada em CSS, responsividade para dispositivos móveis, navegação integrada entre páginas, foco em usabilidade e testes completos de funcionamento.

O desenvolvimento foi realizado sem o uso de JavaScript, uma restrição dentro da documentação das regras propostas. Por isso, funcionalidades dinâmicas como validações avançadas, alterações em tempo real e integrações automatizadas não foram aplicadas.

Análise SWOT

Pontos Fortes: Identidade visual forte, navegação intuitiva e catálogo diversificado.

Pontos Fracos: Ausência de banco de dados real para processamento de vendas.

Oportunidades: Aplicação de ferramentas e práticas próximas ao mercado, aprimorando trabalho em equipe, organização de projeto e desenvolvimento de soluções web.

Ameaças: Adaptação do layout de tabelas para manter o design moderno e periodicidade mensal conforme novos drops de roupas.
