# OUTZ - Streetwear Showroom

Project developed for the Fundamentals of Web Programming course, with the goal of creating a complete website using HTML and CSS.

The project consists of a streetwear fashion e-commerce website, featuring products, detailed product pages, customer registration forms, contact section, and brand information.

## Technologies Used
- HTML5
- CSS3
- Google Fonts
- Responsive Design

## Features
- Homepage with brand presentation
- Product catalog
- Individual product detail pages
- Registration and contact forms
- HTML field validation
- Fully customized layout developed from scratch
- No AI used

## Objective
To practice web development concepts, page structuring, CSS styling, and the creation of a modern browsing experience for an e-commerce website.
